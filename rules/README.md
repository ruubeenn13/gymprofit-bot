# `rules/`

**Español**

Reglas de obligado cumplimiento del proyecto. `coding-rules.md`: Java 21, dominio en español,
i18n obligatorio, embeds solo por `EmbedFactory`, servicios testeables, secrets solo por env,
documentar todo. `review-checklist.md`: lista que se verifica antes de mergear (compila + tests,
sin secretos, ES/EN sincronizados, permisos, cooldown, paleta, docs actualizadas). La plantilla
de PR enlaza la checklist.

_Ejemplo:_ antes de aprobar un PR se recorre `review-checklist.md` punto por punto.

---

**English**

Mandatory project rules. `coding-rules.md`: Java 21, Spanish domain, mandatory i18n, embeds only
via `EmbedFactory`, testable services, secrets only via env, document everything.
`review-checklist.md`: the list verified before merging (build + tests, no secrets, ES/EN in
sync, permissions, cooldown, palette, updated docs). The PR template links the checklist.

_Example:_ before approving a PR, walk `review-checklist.md` point by point.
